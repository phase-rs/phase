**CardGrid.tsx changes:**

1. Replace `getCardImageSmall(card)` with Scryfall art_crop URL. Since CardGrid receives `ScryfallCard` objects with `image_uris`, use `card.image_uris?.art_crop ?? card.card_faces?.[0]?.image_uris?.art_crop ?? ""` to get the art_crop URL directly (no need for the useCardImage hook since we have the Scryfall card data with URLs).

2. Change the image aspect ratio from `aspect-[488/680]` (normal card ratio) to `aspect-[4/3]` (art_crop ratio).

3. Add `object-cover` to the image to handle varying art_crop dimensions.

4. Add a `hoveredCard` state and `onCardHover` callback prop to CardGrid:
   ```typescript
   interface CardGridProps {
     cards: ScryfallCard[];
     onAddCard: (card: ScryfallCard) => void;
     onCardHover?: (cardName: string | null) => void;
   }
   ```
   Each card tile: `onMouseEnter={() => onCardHover?.(card.name)}` and `onMouseLeave={() => onCardHover?.(null)}`.

5. The card name tooltip at bottom should always be visible (remove the translate-y-full hide/show on hover). Instead: always show the card name at the bottom of each tile with `bg-black/70 text-[10px] text-white truncate`.

6. If a card count prop is provided (for deck list view), overlay the count badge at top-right:
   ```typescript
   interface CardGridProps {
     cards: ScryfallCard[];
     onAddCard: (card: ScryfallCard) => void;
     onCardHover?: (cardName: string | null) => void;
     cardCounts?: Map<string, number>;  // for deck list display
   }
   ```
   When `cardCounts` has an entry for a card: show a circular count badge at top-right corner with the count number.
