**StackDisplay.tsx** — Complete rewrite for Arena-style stack visualization:
- Position: The component itself should be styled for right-column placement. Use `fixed right-4 top-1/2 -translate-y-1/2 z-30` positioning. Only render when stack is non-empty (keep `EMPTY_STACK` module-level constant pattern).
- Layout: Stack cards in a staggered pile where each card overlaps the one below. Reverse the stack array for display: `const displayStack = [...stack].reverse()` (engine stack[0] = oldest, but display should show newest on top).
- Each card gets stagger offset: `STAGGER_Y = 28px` per card (enough to show card art below). Add subtle `STAGGER_X = 3px` per card for pile depth illusion.
- Stack card sizing: Inline the sizing logic (no external dependency). Base width 150px, min 80px, shrink when >2 items: `scale = Math.max(0.5, 1 - Math.max(0, stackCount - 2) * 0.083)`. Apply width and height (1.4 aspect ratio) to each card.
- Wrap in `AnimatePresence` with Framer Motion for entry/exit. Container has `backdrop-blur-sm bg-black/40 rounded-xl p-3` for glassmorphism.
- Add a "Stack" header label at top with count badge.
- Use Framer Motion `AnimatePresence` on the whole container (slide in from right when stack goes from empty to non-empty).

**StackEntry.tsx** — Rewrite for full card image preview:
- Accept `entry: StackEntryType`, `index: number`, `isTop: boolean`, and `cardSize: { width: number; height: number }` props.

- For abilities (not spells), render a smaller ability badge overlay at the bottom with the source card name and "Activated" / "Triggered" label.

- Wrap in `motion.div` with `initial={{ opacity: 0, x: 30, scale: 0.9 }}`, `animate={{ opacity: 1, x: 0, scale: 1 }}`, `exit={{ opacity: 0, x: 30, scale: 0.9 }}`.
- Controller indicator: subtle "P1" / "P2" badge at bottom-left.
