  <action>
1. Create `client/src/components/stack/StackEntry.tsx`:
   - Props: `{ entry: StackEntryType }` (from adapter/types.ts).
   - Shows small card image (via CardImage size='small') for spell entries, or ability icon + source name for activated/triggered abilities.
   - Shows controller player indicator.
   - Framer Motion layout animation for stack entries appearing/disappearing.

2. Create `client/src/components/stack/StackDisplay.tsx`:
   - Reads `gameState.stack` from gameStore.
   - Renders StackEntry components in vertical list (top = top of stack, LIFO order).
   - AnimatePresence for entries entering/leaving.
   - Title "Stack" at top. Empty state: subtle "Stack empty" text.
   - Compact vertical layout suitable for side panel.

3. Create `client/src/components/controls/PhaseTracker.tsx`:
   - Reads `gameState.phase` and `gameState.turn_number` from gameStore.
   - Displays all 12 phases as a horizontal row of abbreviated labels (UNT, UPK, DRW, M1, BC, ATK, BLK, DMG, EC, M2, END, CLN).
   - Current phase highlighted with bright color (white text, subtle glow), others dimmed.
   - Turn number display: "Turn {N}".
   - Compact horizontal strip.

4. Create `client/src/components/controls/LifeTotal.tsx`:
   - Props: `{ playerId: number }`.
   - Reads player life from gameStore.
   - Large number display with player label.
   - Framer Motion: animate number changes with a brief scale pulse.
   - Color coding: green when >= 10, yellow when 5-9, red when < 5.

5. Create `client/src/components/controls/PassButton.tsx`:
   - Reads waitingFor from gameStore.
   - When waitingFor is Priority for the current player: show "Pass Priority" button.
   - On click: dispatch GameAction.PassPriority via gameStore.dispatch.
   - Style: prominent button, visible at bottom-right area of board.

6. Create `client/src/components/controls/FullControlToggle.tsx`:
   - Reads/toggles fullControl from uiStore.
   - Toggle button showing "Full Control: ON/OFF".
   - When ON: highlighted in amber color. When OFF: dimmed.
   - Position near PassButton.
