Attack legality -- `pub fn validate_attackers(state: &GameState, attacker_ids: &[ObjectId]) -> Result<(), String>`:
- Each attacker must be a creature on the battlefield controlled by active player
- Must not be tapped (unless it has an ability allowing it)
- Must not have Defender keyword
- Must have Haste keyword OR `entered_battlefield_turn < state.turn_number` (summoning sickness)

Block legality -- `pub fn validate_blockers(state: &GameState, assignments: &[(ObjectId, ObjectId)]) -> Result<(), String>`:
- Each blocker must be a creature on battlefield controlled by defending player (non-active player)
- Each blocker must not be tapped
- Flying attackers can only be blocked by creatures with Flying or Reach
- Menace: if an attacker has Menace, it must be blocked by 2+ creatures or not at all (validate after grouping assignments by attacker)
- Shadow: creatures with Shadow can only be blocked by creatures with Shadow, and non-Shadow creatures can't be blocked by Shadow creatures

`pub fn declare_attackers(state: &mut GameState, attacker_ids: &[ObjectId], events: &mut Vec<GameEvent>) -> Result<(), String>`:
- Validate attackers

`pub fn resolve_combat_damage(state: &mut GameState, events: &mut Vec<GameEvent>)`:
- Check if any creature in combat has FirstStrike or DoubleStrike
- If yes: run first_strike_damage_step first, then regular_damage_step
- If no: run regular_damage_step only
- Between damage steps: run SBAs (sba::check_state_based_actions) and process triggers (triggers::process_triggers)

`fn first_strike_damage_step(state: &mut GameState, events: &mut Vec<GameEvent>)`:
- Only creatures with FirstStrike or DoubleStrike deal damage
- For each such attacker: assign damage to blockers (if blocked) or defending player (if unblocked)
- For each blocker with FirstStrike or DoubleStrike: assign damage to the attacker it's blocking
- Apply all damage assignments via `apply_combat_damage`
- Set CombatState.first_strike_done = true

`fn regular_damage_step(state: &mut GameState, events: &mut Vec<GameEvent>)`:
- Creatures that deal damage: those WITHOUT FirstStrike (they already dealt), plus those WITH DoubleStrike (deal again)
- Same assignment logic as first_strike step
- Apply all damage assignments

`fn assign_attacker_damage(state: &GameState, attacker_id: ObjectId, combat: &CombatState) -> Vec<DamageAssignment>`:
- Unblocked attacker: all damage to defending player
- Blocked by one creature: all damage to that creature
- Blocked by multiple (ordered) creatures with Trample:
  - Assign lethal damage to first blocker, then lethal to second, etc. Lethal = toughness - damage_already_marked. With Deathtouch, lethal = 1 per blocker (MTG 702.2e).
  - Excess damage to defending player
- Blocked by multiple without Trample:
  - Assign lethal to first blocker, then second, etc. Any remaining damage to last blocker (not to player).

`fn apply_combat_damage(state: &mut GameState, assignments: &[(ObjectId, DamageAssignment)], events: &mut Vec<GameEvent>)`:
- For damage to creatures: add to damage_marked. If source has Deathtouch, set dealt_deathtouch_damage on target. Emit DamageDealt event.
- For damage to players: reduce life total. Emit DamageDealt event with TargetRef::Player.
- For each damage dealt: if source has Lifelink, gain that much life for source's controller. Emit LifeChanged event.

`fn lethal_damage(obj: &GameObject, _state: &GameState) -> u32`:
- If deathtouch: return 1 (but caller handles this via source keyword check)
- Otherwise: toughness - damage_marked (minimum 0)
