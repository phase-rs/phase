<objective>
Implement mana abilities per MTG Rule 605 -- activated abilities that produce mana resolve instantly without using the stack.

Purpose: Mana abilities are foundational -- nearly every nonbasic land and mana creature uses them. The current engine has a TapLandForMana shortcut only for basic lands. This plan creates a proper mana ability system that handles any "T: Add {M}" ability and resolves it immediately.

Output: New mana_abilities module, updated engine.rs routing, extended ActivateAbility handling.
</objective>
