  <action>
1. Create `client/src/services/deckParser.ts`:
   - Type: `DeckEntry = { count: number, name: string }`.
   - Type: `ParsedDeck = { main: DeckEntry[], sideboard: DeckEntry[] }`.
   - `parseDeckFile(content: string): ParsedDeck`:
     - Parse .dck/.dec format: "count CardName" per line.
     - Section headers: [Main] [Sideboard] (case-insensitive).
     - Lines starting with # are comments, skip empty lines.
     - Handle both "4 Lightning Bolt" and "4x Lightning Bolt" formats.
   - `exportDeckFile(deck: ParsedDeck): string`:
     - Inverse: generate .dck format string from deck data.

2. Create `client/src/components/deck-builder/CardSearch.tsx`:
   - Search input with debounced Scryfall API calls (300ms debounce).
   - Uses Scryfall search endpoint: `https://api.scryfall.com/cards/search?q=${query}+f:standard` (Standard format filter).
   - Filter controls: color identity buttons (W/U/B/R/G), type dropdown (Creature, Instant, Sorcery, Enchantment, Artifact, Land, Planeswalker), CMC range selector.
   - Constructs Scryfall query string from filters (e.g., `c:r t:creature cmc<=3`).
   - Returns search results as array of Scryfall card objects.
   - Shows result count and loading state.
   - Error handling for rate limits (429) with retry-after respect.

3. Create `client/src/components/deck-builder/CardGrid.tsx`:
   - Props: `{ cards: ScryfallCard[], onAddCard: (card: ScryfallCard) => void }`.
   - Renders cards as a responsive grid of Scryfall images (small size for thumbnails).
   - Each card clickable to add to deck.
   - Shows card name on hover tooltip.
   - Standard legality indicator: green border if Standard-legal, red border + "Not Standard" overlay if not.
   - Block adding non-Standard cards (per locked decision).
   - Framer Motion: cards animate in on search result change.
