  <action>
Create `client/src/components/card/ArtCropCard.tsx` — the battlefield card renderer using art_crop images.

Props: `{ objectId: number }`

**Color border mapping** (per user decision):
```typescript
const BORDER_COLORS: Record<string, string> = {
  W: '#F5E6C8',   // White = cream
  U: '#0E68AB',   // Blue
  B: '#2B2B2B',   // Black = dark
  R: '#D32029',   // Red
  G: '#00733E',   // Green
};
```

Border color logic: If object has 0 colors, use gray (#8E8E8E). If 2+ colors, use gold (#C9B037). If 1 color, map from BORDER_COLORS.

The component:
1. Read the game object from `useGameStore` by objectId
2. Use `useCardImage(obj.name, { size: 'art_crop' })` to get the art crop image
3. Render a container using CSS vars `var(--art-crop-w)` x `var(--art-crop-h)` with rounded-md, overflow-hidden
4. Apply a 3px solid border using the computed border color
5. For tokens: check `obj.is_token` — if true, use 2px border width instead of 3px
6. Render the art crop image with `object-cover` to fill the container
7. Card name label: absolute-positioned at bottom, `bg-black/70 px-1 py-0.5 text-[9px] text-white truncate`
8. P/T box overlay: if `computePTDisplay(obj)` returns non-null, render `<PTBox>` (existing component)
9. Loyalty shield: if `obj.loyalty != null`, render a shield-shaped badge at bottom center — use `bg-gray-900/90 border border-amber-400 rounded-full px-2 py-0.5 text-xs font-bold text-amber-300`
10. Counter badges: render as circular overlays at top-right. Each counter: `rounded-full w-5 h-5 flex items-center justify-center text-[9px] font-bold text-white` with the existing COUNTER_COLORS mapping (Plus1Plus1=green-600, Minus1Minus1=red-600, Loyalty=amber-600, default=purple-600)
11. Loading state: show a shimmer placeholder at the same size as the art crop container
12. Unimplemented mechanics badge: if `obj.has_unimplemented_mechanics`, show small amber "!" badge at top-left (same as existing CardImage behavior)

Import `computePTDisplay` from `../../viewmodel/cardProps.ts` and `PTBox` from `../board/PTBox.tsx`. Import `useCardImage` from `../../hooks/useCardImage.ts`. Import `useGameStore` from `../../stores/gameStore.ts`.
